# Description of the Compiler #

## Comment Consistency Checker ##

Implements the comment and note consistency checker. 

@[file:stz-comments.stanza] The implementation of the algorithm. 
@[file:stz-comments-main.stanza] A top-level driver for calling the algorithm with appropriate command-line flags.
@[file:stz-comments-tests.stanza] A REPL testing harness for trying out individual algorithm passes.

## Mocker System ##

Implements the mocker system for easy creation of testing harnesses.

@[file:stz-mocker.stanza] The implementation of the `MockScript` utility.
