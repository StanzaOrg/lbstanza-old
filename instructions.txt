# Developing the trampoline #

Run this in the REPL:

  load stz/test-c-extern-trampoline

Generates this file:

  c-extern-code.s

Compile and run the driver:

  gcc compiler/test-c-extern-driver.c c-extern-code.s -o test-c-extern-driver
  ./test-c-extern-driver

# Bootstrapping #

Build the bootstrap compiler:

  stanza build -verbose -flags BOOTSTRAP -pkg bootpkgs -o stanzaboot

Build the regular system:

  ./stanzaboot build -verbose -pkg pkgs
