How to bootstrap:

Make sure Stanza is updated, and delete the current stanza.aux file
if necessary.

    stanza clean
    stanza version

Should print out:

                 L.B.Stanza Programming Language

                        Version 0.17.28

    Copyright (c) 2016-2019, Patrick Shaobai Li, The Regents of the 
    University of California. All Rights Reserved.

Run this:

    stanza build -optimize -verbose -flags BOOTSTRAP

This builds this executable:

    ./stanza

Run this to build the debugger:

    ./stanza core/stanza.proj scratch/debugger.stanza -o debugger

Run this to build an example program:

On OS-X:

    ./stanza core/stanza.proj scratch/print-safepoints.stanza -o print-safepoints

On Linux:

    ./stanza core/stanza.proj scratch/print-safepoints.stanza -o print-safepoints -ccflags "-shared"

Run the debugger:

    ./debugger ./print-safepoints
    
Type in the following commands:

    break "scratch/print-safepoints.stanza" 8
    run
    continue
    continue
